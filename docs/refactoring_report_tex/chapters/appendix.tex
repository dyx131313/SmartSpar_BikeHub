\subsection*{已知限制}

\begin{itemize}
  \item 高德地图依赖网络、Key 和安全码，本地不可用时会自动切换为站点分布图。
  \item 前端 lint 当前为 0 errors，但仍保留历史 warning，后续可按模块逐步清理。
  \item 重新训练深度学习预测模型需要单独准备 torch 相关环境；本次验收重点为已生成预测数据的读取和展示。
\end{itemize}
