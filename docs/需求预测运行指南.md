# 需求预测运行指南

## 模型介绍

### DLinear (Decomposition Linear)

架构类型：Linear (线性模型)

特点：

- 极度轻量：它本质上只是几个全连接层（Linear Layers），参数量极少，训练和推理速度最快。
- 基准之王：在许多长序列预测任务中，它击败了复杂的 Transformer 模型。
- 原理适配：它内置了“趋势-季节性”分解（Trend-Seasonality Decomposition），非常适合处理共享单车这种既有早晚高峰（季节性）又有总体增长或下降（趋势）的数据。
- 文件位置：models/DLinear.py

### TiDE (Time-series Dense Encoder)

架构类型：MLP (多层感知机)

特点：

- 协变量处理能力强：TiDE 的设计初衷就是为了更好地融合静态协变量（如站点类型）和动态协变量（如天气、温度、节假日）。数据中有丰富的外部特征，TiDE 能很好地利用它们。
- 推理速度快：作为纯 MLP 架构，没有复杂的注意力机制，推理延迟极低，非常适合 Web 后端实时调用。
- 文件位置：models/TiDE.py

### TimesNet

架构类型：CNN + FFT (卷积神经网络 + 傅里叶变换)

特点：

- 捕捉多周期性：共享单车数据具有复杂的周期（日周期、周周期）。TimesNet 通过将一维时间序列转化为二维张量，利用 2D 卷积捕捉这种“多周期”变化，效果通常优于普通模型。
- 综合性能好：它是 TSlib 仓库中的明星模型之一，在多变量预测任务上表现非常稳健。虽然比 DLinear 稍重，但在 GPU 上依然非常快。
- 文件位置：models/TimesNet.py

#### 对比表

|        | DLinear    | TiDE        | TimesNet    |
| ------ | ---------- | ----------- | ----------- |
| 速度   | 5s / epoch | 15s / epoch | 50s / epoch |
| 参数量 | 轻量       | 中等        | 中等        |

## 环境配置

- python >= 3.10

requirements.txt：

```cmd
numpy==1.23.5
pandas==1.5.3
scikit_learn==1.2.1
torch==2.3.1+cu121
tqdm==4.64.0
```

## 运行方法

```cmd
cd backend\app\services\time_series

python train.py --model DLinear
python train.py --model TiDE
python train.py --model TimesNet
```
