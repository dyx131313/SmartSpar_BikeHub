## 4. 启动命令（端口冲突处理）

### ① 先看谁在占用 5000
```bash
lsof -i :5000
```
- 若显示 `python run.py` + 别人的 UID → **勿杀**，用方案 ②  
- 若显示 `python run.py` + **你自己** → 可 `kill -9 <PID>` 再重启

### ② 换端口启动
-先在run.py底部更改 app.run(debug=False, host='0.0.0.0', port=5000)，将5000改成别的端口
```bash
python run.py 0.0.0.0 xxxx            #换成任意空闲端口
```
访问地址：`http://123.60.142.229:xxxx`
