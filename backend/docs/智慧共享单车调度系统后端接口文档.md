# 智慧共享单车调度系统后端接口文档

## 项目信息

- **项目名称**: 智慧共享单车调度系统 (SmartSpar BikeHub)
- **API版本**: v1.0
- **基础URL**: `http://localhost:5000/api`
- **技术栈**: Python Flask + MySQL + JWT认证

## 通用信息

### 请求格式
- **Content-Type**: `application/json`
- **认证方式**: Bearer Token (JWT)
- **字符编码**: UTF-8

### 响应格式
所有接口统一返回JSON格式：

#### 成功响应格式
```json
{
  "message": "操作成功",
  "data": {
    // 具体数据
  },
  "total": 100,        // 仅分页接口
  "pages": 5,          // 仅分页接口
  "current_page": 1,   // 仅分页接口
  "per_page": 20       // 仅分页接口
}
```

#### 错误响应格式
```json
{
  "error": "错误描述信息"
}
```

### 认证说明
除登录接口外，所有接口都需要在请求头中携带JWT Token：

```
Authorization: Bearer <your_jwt_token>
```

## 需求数据管理

### 1. 获取需求数据

**接口地址**: `GET /api/demand-data`

**查询参数**:
- `page` (可选): 页码，默认1
- `per_page` (可选): 每页数量，默认20，最大100
- `station_type` (可选): 站点类型过滤
- `start_date` (可选): 开始日期，格式：2025-11-01T00:00:00
- `end_date` (可选): 结束日期，格式：2025-11-12T23:59:59
- `weather` (可选): 天气过滤
- `weekday` (可选): 星期几(1-7)

**响应数据**:
```json
{
  "data": [
    {
      "id": 1,
      "timestamp": "2025-07-01T08:00:00",
      "station_id": 1,
      "station_type": "canteen",
      "weekday": 2,
      "is_holiday": false,
      "weather": "rain",
      "temp": 28.4,
      "demand": 14,
      "created_at": "2025-11-12T09:00:00"
    }
  ],
  "total": 100,
  "pages": 5,
  "current_page": 1,
  "per_page": 20,
  "has_next": true,
  "has_prev": false
}
```

  - `"用户名已存在"`

## 数据类型说明

### 用户角色 (role)
- `admin`: 管理员
- `dispatcher`: 调度员

- `user`: 普通用户

### 站点类型 (station_type)

- `teaching_building`: 教学楼
- `dormitory`: 宿舍区

### 任务状态 (status)
- `pending`: 待处理
- `in_progress`: 进行中
- `completed`: 已完成
- `cancelled`: 已取消

- `sunny`: 晴天
- `cloudy`: 多云
- `rain`: 雨天

- `windy`: 大风

## 使用示例

### 完整的调度流程示例

1. **登录获取Token**
```bash
curl -X POST http://localhost:5000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"dispatcher01","password":"password123"}'
```

2. **创建调度任务**

```bash
curl -X POST http://localhost:5000/api/dispatch-tasks \
```

```bash
curl -X GET http://localhost:5000/api/dispatch-tasks/11 \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```bash
curl -X POST http://localhost:5000/api/predictions/batch \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -d '{
    "station_ids": [1, 2],
    "start_time": "2025-11-13T00:00:00",
    "end_time": "2025-11-13T23:00:00"
  }'
```

## 注意事项

1. **Token过期时间**: 24小时，过期后需要重新登录
2. **分页限制**: 每页最大返回100条记录
3. **文件上传限制**: 最大16MB
4. **数据保留**: 历史数据保留7天，支持快速检索
5. **并发支持**: 支持50个用户同时交互，10个调度员在线操作
6. **路径规划**: 支持途径点数≥10，并能选出最优解
